## Getting Started

 * [Ansible project](https://github.com/ansible/ansible) -- see the examples directory


## Quick setup 
```
git clone https://github.com/luke-h1/Automation 
cd ~/Automation
Run a playbook = ansible-playbook --ask-pass -K -K -vvv -u <your-username> <playbook-name>
```
